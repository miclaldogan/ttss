# Changelog

All notable changes to TTSS will be documented in this file.

The format follows Keep a Changelog and the project uses Semantic Versioning.

## [0.1.0] - 2026-05-23

### Added

- Initial TTSS repository scaffold with README, contribution guide, license, and CI workflow.
- Core package layout covering data, models, training, inference, evaluation, scripts, and tests.
- Temporal labeling and preprocessing pipeline for UCF-Crime, including prepared JSON annotations.
- PyTorch implementation of the TTSS three-tier threat scoring architecture.
- Documentation for the architecture, early-warning flow, and temporal labeling scheme.
- Developer automation via Makefile targets for install, lint, test, train, demo, and cleanup.
